## 🚀 Deploy Now!

### **Option 1: GitHub Pages (Recommended - FREE)**

```bash
# Navigate to your portfolio folder
cd /path/to/portfolio/files

# Run the automated script
chmod +x deploy-github.sh
./deploy-github.sh

# Follow the prompts:
# - Enter GitHub username: ferdnand
# - Enter repository name: portfolio (or ferdnand.github.io)
```

**Your site will be live at:**
- `https://ferdnand.github.io/portfolio/`
- **OR** `https://peakdatastories.com` (after DNS setup)

---

### **Option 2: Manual GitHub Upload**

```bash
# Initialize git
git init
git add index.html publications.html experience.html resume-tool.html error.html

# Commit
git commit -m "Complete portfolio with AI resume tool"

# Push to GitHub (create repo first at github.com/new)
git remote add origin https://github.com/ferdnand/portfolio.git
git branch -M main
git push -u origin main

# Enable GitHub Pages in repository settings
```

---

### **Option 3: S3**

```bash
# Upload all files
aws s3 cp index.html s3://YOUR-BUCKET/
aws s3 cp publications.html s3://YOUR-BUCKET/
aws s3 cp experience.html s3://YOUR-BUCKET/
aws s3 cp resume-tool.html s3://YOUR-BUCKET/
aws s3 cp error.html s3://YOUR-BUCKET/
```
